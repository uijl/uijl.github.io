The first step we want to do is update and upgrade our subsystem. Open bash and use the command below. root@host: path# sudo apt update && upgrade Once your system is up-to-date you can install the python-virtual environment. root@host: path# apt-get install python-virtualenv Windows Subsystem for Linux comes installed with two python versions installed, before we continue you need to decide which Python distribution you want to use for your virtual environment.
